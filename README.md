### Software
- **OS:** Ubuntu24.04

### Usage

1.  **Start Server**
    Run on the first machine:
    ```bash
    python3 server.py
    ```

2.  **Start Proxy**
    Run on the second machine:
    ```console
    ./final_proxy
    Server IP: <SERVER_IP_ADDRESS>
    ```

3.  **Start Client**
    Modify the IP in `client.py` and run on the third machine:
    ```bash
    python3 client.py
    ```
